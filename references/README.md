This directory `~/references` contains the reference files for book, articles and links.
They are produced by the script `~/build.py` from the following sources:

- `~/source/goodreads_library_export.csv` CSV file exported from Goodreads.
- `~/source/corrections` Corrections to the Goodread entries. File name is the Goodread identifier. Format is YAML. Any entry may be changed.
- `~/source/references` Articles, link and books (not in Goodreads) added manually. Format is YAML.
